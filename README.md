⚡ Smart Energy Monitoring Dashboard

A web-based dashboard for analyzing and visualizing household energy consumption using Python, Streamlit, Pandas, and Plotly.

🚀 Live Demo

🔗 Live Application:(https://smart-energy-dashboard-m82tfvnaspq94we5ovz7vi.streamlit.app/)

📖 Project Overview

The Smart Energy Monitoring Dashboard helps visualize electricity consumption data through interactive charts and statistical insights. It demonstrates how data analytics can be used to understand energy usage patterns and build user-friendly dashboards.

✨ Features

- 📊 Interactive energy consumption dashboard
- ⚡ Power consumption analysis
- 🔋 Voltage trend visualization
- ⚙️ Current (Global Intensity) analysis
- 🏠 Sub-metering comparison
- 📈 Statistical summary of the dataset
- 🧹 Automatic data cleaning

🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- NumPy
- Scikit-learn
- Git & GitHub

🚀 Installation

Clone the repository:

git clone https://github.com/jagtapmansis006-glitch/Smart-Energy-Dashboard.git

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
